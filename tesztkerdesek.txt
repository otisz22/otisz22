Kérdés 4:
Milyen eszközökkel jelenítheti meg a Kanban módszer a munkafolyamatokat?

A. Gantt táblázatokkal
B. Word dokumentumokkal
C. Kanban táblával ✔
D. PowerPoint prezentációkkal

Válasz: a helyes válasz a C. azaz Kanban táblával
A Kanban tábla egy vizuális eszköz, amelyen az oszlopok és a feladat kártyák segítségével ábrázolják a munkafolyamatot.
A Gantt a projektmenedzsmentben gyakran használt eszközök, de nem specifikusak a Kanban módszerhez. A Gantt táblázatok kronológiai áttekintést adnak a feladatok időzítéséről és az egymáshoz való kapcsolódásról.
A Word dokumentumok és a PowerPoint prezentációk inkább dokumentáció és prezentáció készítésére szolgálnak, és kevésbé alkalmasak a dinamikus és vizuális munkafolyamatok ábrázolására.

Kérdés 9:
Mi a célja a git clone parancsnak?

A. Egy meglévő git repository másolatának létrehozása a helyi rendszeren
B. Egy új git repository létrehozása
C. A módosítások összefésülése a git repository-ban
D. A git repository törlése

Válasz:
A hleyes válasz: A. Egy meglévő git repository másolatának létrehozása a helyi rendszeren
Miután a "git clone" parancsot használtuk, a helyi gépen már lehetőségünk van a repositoryon dolgozni, változtatásokat végrehajtani, új commit-okat létrehozni és a verziókezelő funkciókat használni.
git clone <repository-url>


Kérdés 10:
Melyik az alábbiak közül egy bevett clean code gyakorlat?

A. Az egyes metódusok és függvények hosszú és összetett nevekkel való ellátása, hogy minden részletet leírjanak
B. A változók, metódusok és osztályok beszédes neveinek használata, amelyek tükrözik a funkciójukat és a szerepüket
C. A kód duplikálása, hogy biztosítsa a függetlenséget az egyes modulok között
D. A metódusok és függvények minél nagyobbá tétele a lehető legtöbb logika beépítése érdekében

Válasz:
A helyes válasz: B. A változók, metódusok és osztályok beszédes neveinek használata, amelyek tükrözik a funkciójukat és a szerepüket
A clean code gyakorlata szerint a kód olvashatóságának javítása érdekében fontos, hogy a változók, metódusok és osztályok nevei pontosan és beszédesen tükrözzék a funkciójukat és szerepüket.
